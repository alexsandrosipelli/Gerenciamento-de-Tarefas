Sobre o Projeto

Este projeto é um Sistema de Gerenciamento de Tarefas desenvolvido com tecnologias avançadas como Spring Boot, Java, H2, HTML, CSS, Thymeleaf, e JavaScript. Ele foi criado para proporcionar uma maneira eficiente e organizada de gerenciar tarefas, oferecendo funcionalidades completas que atendem às necessidades dos usuários de forma prática e intuitiva.

  • Criação de usuários com validação de formulários.
  • Adição, edição e remoção de tarefas.
  • Listagem de tarefas com opções de filtragem.
  • Integração com banco de dados H2.
  • Interface de usuário responsiva e amigável.
