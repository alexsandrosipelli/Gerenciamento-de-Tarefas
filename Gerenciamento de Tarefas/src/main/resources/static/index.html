Bem-vindo ao Sistema de Gerenciamento de Tarefas

Sobre o Projeto

Este projeto é um Sistema de Gerenciamento de Tarefas desenvolvido utilizando Spring Boot, Java, H2, HTML, CSS, JavaScript. O objetivo do projeto é permitir que os usuários gerenciem suas tarefas de forma eficiente e organizada.

  • Criação de usuários com validação de formulários.
  • Adição, edição e remoção de tarefas.
  • Listagem de tarefas com opções de filtragem e ordenação.
  • Integração com banco de dados H2.
  • Interface de usuário responsiva e amigável.
